# PDF Scripts

This was written to identify italicized strings in a very large (4500+ pages) ebook in PDF format.

# Capabilities

- Distinguish between footnotes and main body of text
- Match footnote numbers
- Identify all italicized strings, but do not overextend into non-italicized strings

# Aims

- Formatted epub document with nice footnotes and italicized text preserved since the author heavily utilizes italics in order to convey meaning and format Latin terms
- 99% accuracy
